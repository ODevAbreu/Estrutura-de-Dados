# 📚 Estrutura de Dados em Java

Repositório dedicado ao estudo e implementação de **estruturas de dados** utilizando Java.
Este projeto tem como objetivo consolidar conceitos fundamentais da computação, com foco em organização, desempenho e boas práticas de programação.

---

## 🚀 Tecnologias utilizadas

* Java ☕
* Programação Orientada a Objetos (POO)
* Estruturas de Dados clássicas

---

## 📂 Estruturas implementadas

* 🌳 Árvore Binária de Busca (BST)
* 📋 Lista Encadeada
* 📚 Pilha (Stack)
* 📦 Fila (Queue)
* 🔗 Outras estruturas em desenvolvimento...

---

## 🌳 Exemplo: Árvore Binária de Busca

A BST segue a regra:

* Valores menores ficam à esquerda
* Valores maiores ficam à direita

```java
ArvoreBinariaBusca arvore = new ArvoreBinariaBusca();

arvore.inserir(10);
arvore.inserir(5);
arvore.inserir(15);
arvore.inserir(3);
```
